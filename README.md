# Study Tracker

A beginner-friendly study tracking app built with React, TypeScript, and Vite.

## Getting started

You need a current Node.js LTS release and npm installed.

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Available scripts

- `npm run dev` starts the development server.
- `npm run build` checks TypeScript and creates a production build.
- `npm run typecheck` checks TypeScript without creating files.
- `npm run preview` previews the production build locally.
